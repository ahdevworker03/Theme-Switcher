# 🌗 React Theme Switcher

A simple React application that allows users to toggle between Light and Dark themes using the **useContext** hook for state management.

## 🚀 Live Demo

[View Live Demo](https://ahdevworker03.github.io/Theme-Switcher/)

## ✨ Features

- 🌞 Light / 🌙 Dark mode toggle
- 🔄 Smooth CSS transitions
- 📦 State management with React Context API
- ⚡ Built with Vite for fast development

## 🛠️ Tech Stack

- React
- Vite
- CSS3
- Context API

## 📁 Project Structure

```
src/
├── context/
│   └── ThemeContext.jsx
├── components/
│   └── ThemeSwitcher.jsx
├── App.jsx
├── App.css
└── main.jsx
```

## 🏃‍♂️ Run Locally

```bash
# Clone the repository
git clone https://github.com/ahdevworker03/Theme-Switcher.git

# Navigate to project
cd Theme-Switcher

# Install dependencies
npm install

# Start development server
npm run dev
```

## 📦 Build for Production

```bash
npm run build
```

## 🚀 Deploy to GitHub Pages

```bash
npm run deploy
```
